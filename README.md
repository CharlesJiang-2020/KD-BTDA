KD-BTDA-Pytorch
=
这是我们的论文的 GitHub 存储库：面向自闭症辅助诊断的知识蒸馏混合域适应方法

Requirements:
=
torch == 1.8.1+cu111  
torch-cluster           1.5.9  
torch-geometric         1.6.0  
torch-scatter           2.0.8  
torch-sparse            0.6.12  
sklearn

ABIDE I dataset
=
我们使用了由 CPAC 预处理的 ABIDE I 数据集，请在官网下载：http://fcon_1000.projects.nitrc.org/indi/abide/abide_I.html

运行：
=
'<python KD-BTDA.py>'
